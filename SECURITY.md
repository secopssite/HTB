# Security Policy

## Purpose

This repository contains Hack The Box (HTB) writeups and exploit scripts for **educational and authorized security training purposes only**. All content was developed in controlled lab environments.

## Reporting Security Issues

If you discover a security issue in this repository (e.g., accidentally committed real-world credentials, exposed sensitive data, or a vulnerability in any included scripts), please report it responsibly:

1. **Do NOT open a public issue.**
2. Contact via GitHub Security Advisories on this repository.
3. Alternatively, open a private vulnerability report via GitHub's security tab.

## Scope

- Scripts and exploits in this repository are intended for use **only** in authorized environments (HTB, CTF competitions, personal labs).
- Do not use any tools or techniques from this repository against systems you do not own or have explicit authorization to test.

## Credential Handling

Credentials appearing in writeups are from ephemeral HTB lab instances and are not valid outside those environments. If you believe real credentials were accidentally included, please report it immediately.

## Disclaimer

Unauthorized use of techniques described in this repository against systems without explicit permission is illegal. The repository maintainer assumes no liability for misuse.
