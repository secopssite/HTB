# SEO Checklist for GitHub Global Search

✓ Full machine names and variations in README
✓ Complete flags in searchable format (without HTB{} prefix for challenges)
✓ CVE numbers mentioned where applicable
✓ Tool names in every writeup
✓ Technique names (ACL abuse, RBCD, Golden Ticket, etc.)
✓ Code snippets that search engines can index
✓ Hostname keywords in backticks (garfield.htb, dc01.garfield.htb)
✓ File names match search keywords (garfield.md, not just writeup.md)
✓ Topics cover all major categories
